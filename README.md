## Install
Create environment
```
python -m venv .venv
source .venv/bin/activate
```

Install dependencies
```
pip install -r requirements.txt
```

Configure the Enviroment variables. For development stage, you can use an .env file:

```
cp .env.example .env
```

then fill each variable with its true value ;)
